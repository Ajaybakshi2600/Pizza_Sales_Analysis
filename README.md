# Pizza_Sales_Analysis
Data analytics project showcasing Pizza sales analysis using SQL and Power BI.\
This project showcases a complete ETL (Extract, Transform, Load) data pipeline, demonstrating proficiency in both SQL and Power BI for actionable business intelligence.\
Deliverables:\
1.Import data in SQL Serverand Data Analysis (SQL) : import the data in SQL server ,Organize the data into a structured format, simulate business transactions, and execute queries to calculate the KPI metrics needed to extract insights on Pizza Sales performance with respect to revenue, quantity and Orders and to understand the daily and monthly trend on Sales.\
2. Visualization & Insights (Power BI): Connect Power BI to SQL server , clean the data using Power query, create new DAX Measures and build an interactive dashboard that highlights key patterns and trends according to chart requirements, enabling stakeholders to make data-driven decisions.\
3. Report and Presentation: Write a clear project report summarizing your key findings
and business recommendations. Prepare a presentation that visually communicates
insights and actionable recommendations to stakeholders.\
4. GitHub Repository: Include all CSV files, SQL queries, and dashboard files in a
well-structured repository.\
This project validates my ability to manage data end-to-end, from raw input to extracting valuable, actionable insights.
